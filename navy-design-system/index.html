Navy Design System

Kompletny system designu frontend w eleganckich odcieniach granatu. Profesjonalna typografia, komponenty UI i spójna paleta kolorów.

Główne Cechy Systemu

Granatowa Paleta

Profesjonalna paleta kolorów oparta na odcieniach granatu z akcentami niebieskiego.

Tailwind CSS

Zbudowany na Tailwind CSS z rozszerzoną konfiguracją i custom utilities.

Komponenty UI

Bogata biblioteka gotowych komponentów: przyciski, karty, nawigacja i więcej.

Wydajność

Zoptymalizowany pod kątem szybkości ładowania i płynnych animacji.

Responsive

W pełni responsywny design dostosowany do wszystkich urządzeń.

Dostępność

Zgodny z WCAG 2.1, zapewniający dostępność dla wszystkich użytkowników.

Jak Używać

System designu Navy jest gotowy do natychmiastowego użycia w Twoich projektach. 

# Instalacja zależności
npm install tailwindcss @fontsource/inter @fontsource/poppins

# Konfiguracja Tailwind
npx tailwindcss init

# Import stylów
@import 'css/global.css';

# Użycie komponentów
<button class="btn-primary">Przycisk</button>
<div class="card">Zawartość karty</div>
